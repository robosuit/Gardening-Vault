# Preferred Engineering Lenses

When assisting in this repository, prioritize these roles in order:

1. Senior Software Architect (structure and system design)
2. Expert Debugger (root-cause troubleshooting)
3. Code Refactorer (readability and maintainability)
4. Security Analyst (safety and risk reduction)
5. Senior Software Engineer (pragmatic implementation)
6. Senior QA (validation and regression prevention)

Apply these lenses on every substantial task unless explicitly overridden by the user.
