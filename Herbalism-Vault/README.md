# 🌿 Herbalism Vault

**Zone:** 7B  
**Purpose:** Medicinal plant cultivation + apothecary + companion design + graph-based herbal knowledge system.

This vault connects:
- Plants
- Medicinal actions
- Months
- Garden placement
- Energetics
- Preparation methods

## 🔗 How to Use This Vault

### 1. **Graph View**
Open Graph View to explore clusters:
- Digestive herbs cluster
- Respiratory support network
- Pollinator plants
- Mediterranean dry-soil guild

### 2. **Start Here**
Begin with index files in `00_Indexes/`:
- **Master Plant Index** - See all plants by annual/perennial
- **Medicinal Actions Index** - Browse by health function
- **Month-by-Month Calendar** - Plan your growing season
- **Color & Energetics Index** - Explore plant energetics

### 3. **Plant Files**
Each plant in `01_Plants/` has:
- Growing conditions
- Harvest timing
- Medicinal actions
- Preparation methods
- Companion plants

### 4. **Garden Design**
In `04_Garden_Design/`:
- Height mapping
- Water zone planning
- Companion guilds
- Bed layouts

### 5. **Preparations**
In `03_Preparations/`:
- Tea infusion ratios
- Tincture recipes
- Salve formulas
- Syrup guides

### 6. **Monthly Planning**
In `05_Months/`:
- What to start/sow each month
- Zone 7B frost dates
- Succession planting

## 🏷️ Tag System

Use these tags for graph filtering:

**Medicinal:** `#digestive` `#nervine` `#respiratory` `#antimicrobial` `#wound` `#immune` `#circulatory`

**Growth:** `#annual` `#perennial` `#sun` `#partial-shade` `#low-water` `#moist-soil`

**Color:** `#purple` `#yellow` `#white` `#green`

**Energetics:** `#warming` `#cooling` `#drying` `#moistening`

## 📊 Graph Architecture

Plants automatically link to:
- Months (planting calendar)
- Medicinal categories (action-based)
- Companion guilds (growth synergy)
- Color index (visual organization)
- Preparation methods (usage)

## 🚀 Getting Started

1. Open this folder in **VS Code**
2. Install Obsidian and enable **Graph View**
3. Start with `Master Plant Index`
4. Follow monthly planting guides in `05_Months/`
5. Cross-reference preparations and medicinal uses
6. Use tags to discover plant clusters
