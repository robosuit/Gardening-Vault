# Gourd

## Quick Use Guide
- Seed depth: 1 in
- Spacing: rows 72-96 in, plants 24-36 in
- Feeding level: Heavy
- Companion group: Cucurbit

## Planting Calendar (Zone 7B)
- Start window: Apr-Jun
- Indoor start method: Start indoors 3-4 weeks before transplanting or direct sow warm soil.
- Harvest window: Sep-Oct

- Days to germination: 7-10 days

- Bloom speed tier: Fast
