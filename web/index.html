How To Use

  1. Pick a vault on the left to narrow data to herbs, vegetables, fruits, wildflowers, or indexes.
  2. Use the center rail for quick modes (`Plants`, `Indexes`, `Guides`) and to switch `Cards` or `List` view.
  3. Use search, type filter, and sort controls to organize results exactly how you need.
  4. Click any result once to select it, then use actions on the right or double-click to open full note content.
  5. Run npm run build-data after markdown edits so GitHub Pages always reflects current vault data.
